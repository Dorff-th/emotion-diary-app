# 🎨 Frontend - Vue.js

## 🛠️ 기술 스택
- Vue 3 (Composition API)
- Vite
- Tailwind CSS
- Vue Router
- Axios

## ▶️ 실행 방법
```bash
cd frontend
npm install
npm run dev
```

## 📂 주요 디렉토리

- `components/` – 재사용 UI 구성요소
- `views/` – 화면 단위 컴포넌트
- `api/` – 백엔드 통신 모듈
- `router/` – 라우팅 설정
- `layouts/` – 사용자/관리자 레이아웃 구분

## 🔐 인증 방식
- JWT 토큰을 `localStorage`에 저장
- 로그인 시 라우팅 분기 처리 (`USER`, `ADMIN`)
